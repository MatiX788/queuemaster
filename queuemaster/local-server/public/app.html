📋 Installation auf dem Tablet
  1. APK mit dem Button oben herunterladen
  2. In der Benachrichtigungsleiste auf die heruntergeladene Datei tippen
  3. Falls eine Warnung kommt: „Unbekannte Apps zulassen" erlauben
  4. Auf „Installieren" tippen
  5. App öffnen → Server-URL eingeben (steht im Server-Fenster auf dem PC)
⚠️ APK noch nicht verfügbar?
Die APK wird automatisch über GitHub Actions gebaut. Siehe APK_BAUEN.md für Anleitung.
