# Downloads-Ordner

Hier wird die fertige `QueueMaster.apk` abgelegt, nachdem sie von GitHub Actions gebaut wurde.

## APK erhalten:

1. Siehe `APK_BAUEN.md` im Projekt-Hauptordner
2. APK von GitHub herunterladen
3. Hier als `QueueMaster.apk` ablegen
4. Server neu starten

Im Admin unter „Tablet-App" ist sie dann automatisch verfügbar und kann vom Tablet heruntergeladen werden.
